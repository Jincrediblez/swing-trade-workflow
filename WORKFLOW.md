## 第1步：宏观与行业趋势定方向

**目标**：判断当前大盘周期 + 科技/成长主线的位置，确定"现在该看什么方向"

### 核心问题清单

- [ ] 大盘处于什么阶段？（牛市初期/中期/末期，或震荡）
- [ ] 当前资金主线在哪里？（AI、半导体、云计算、新能源、生物科技？）
- [ ] 利率环境对成长股友好吗？（10年期美债收益率、降息预期）
- [ ] 半导体周期处于什么位置？（触底/复苏/繁荣/过热）
- [ ] 是否有重大政策/事件催化？（关税、补贴政策、地缘冲突）

### 关键数据来源

| 数据 | 来源 | 频率 |
|------|------|------|
| 宏观数据（CPI、就业、利率）| FRED / 东方财富 | 每月 |
| 大盘趋势（SPY/QQQ）| TradingView / Futu | 每日 |
| 板块轮动 | Sector ETFs (XLK, SMH, etc.) | 每日 |
| 热点发现 | 市场新闻 / 社区讨论 | 实时 |

## 第2步：初筛股票池

**目标**：从数千只标的中快速缩小到 20-50 只候选

### 筛选维度

| 维度 | 条件示例 | 权重 |
|------|---------|------|
| **成长性** | 营收增速 > 25%，净利润增速加速 | ⭐⭐⭐⭐⭐ |
| **市值** | 大盘科技为主（>50B），可搭配高成长中小盘（10-50B） | ⭐⭐⭐⭐ |
| **技术形态** | 接近关键均线、刚突破整理平台、或回踩关键支撑 | ⭐⭐⭐⭐ |
| **相对强度** | RS Rating > 75，板块内排名前 20% | ⭐⭐⭐ |
| **机构持股** | 机构数量增加、知名基金重仓 | ⭐⭐⭐ |
| **业绩预期** | 下季度业绩预期上调 | ⭐⭐⭐⭐ |

### 初筛工具

- **选股器**：使用自然语言或条件筛选
- **板块扫描**：先定板块，再选龙头
- **动量扫描**：近期创新高/突破的标的

## 第3步：基本面深度分析（超级成长 + 拐点验证）

**目标**：确认这家公司是不是"真成长"、是不是处于拐点

### 分析框架

#### 3.1 业绩拐点信号

| 指标 | 检查项 | 拐点信号 |
|------|--------|---------|
| **营收增速** | 连续 2-3 个季度加速？ | 是 → 强烈信号 |
| **毛利率** | 环比/同比改善？ | 是 → 盈利能力修复 |
| **运营利润率** | 拐点向上？ | 是 → 经营杠杆释放 |
| **自由现金流** | 转正或大幅改善？ | 是 → 财务质量提升 |
| **业绩超预期** | 连续 beat 预期？ | 是 → 市场预期重塑 |

#### 3.2 成长驱动力

- **TAM（市场规模）**：核心业务的市场空间还有多大？
- **新产品/新市场**：贡献节奏如何？
- **定价权**：能否涨价？客户接受度？
- **竞争格局**：市场份额变化趋势？

#### 3.3 竞争壁垒

- 技术壁垒（专利、know-how）
- 网络效应（平台、生态）
- 转换成本（客户粘性）
- 规模效应（成本优势）

#### 3.4 估值合理性

对于超级成长股：
- 重点看 **PEG、P/S、EV/Revenue**，而不是单纯 PE
- 与历史估值区间对比
- 与同业对比（横向估值）
- 结合增速看估值（增速是否匹配估值）

## 第4步：技术面精准分析（拐点确认）

**目标**：找到"基本面改善"与"技术面突破/企稳"的共振点

### 关键看点

#### 4.1 趋势结构

| 条件 | 买入信号 | 说明 |
|------|---------|------|
| 股价位于 50日/200日均线上方 | ✅ | 中期趋势向上 |
| 均线多头排列（5>20>50>200） | ✅ | 强趋势确认 |
| 刚突破长期整理平台 | ✅ | 爆发力最强 |
| 回踩关键均线后企稳 | ✅ | 风险收益比最佳 |

#### 4.2 量能配合

- 突破关键位置时成交量放大（>50日均量 1.5 倍）
- 调整时缩量（<50日均量 70%）
- 量价同步 = 健康；量价背离 = 警惕

#### 4.3 动量指标

| 指标 | 买入信号 | 卖出/警惕信号 |
|------|---------|--------------|
| **RSI** | 从超卖区（<30）回升 | >80 严重超买 |
| **MACD** | 金叉或底背离 | 顶背离 |
| **KDJ** | 低位金叉（<20） | 高位死叉（>80）|
| **BOLL** | 收窄后向上突破 | 突破上轨后回落 |

#### 4.4 支撑阻力位

- **前高/前低**：历史关键价格点
- **斐波那契回撤**：38.2%、50%、61.8%
- **筹码密集区**：成交量分布图（Volume Profile）
- **整数关口**：$50、$100 等心理价位

## 第5步：资金面 + 情绪面共振验证

**目标**：确认"聪明钱"也在进场，市场情绪不是过热

### 5.1 资金异动

| 信号 |  bullish | bearish |
|------|---------|---------|
| 大单净流入 | ✅ | |
| 主力资金连续流入 | ✅ | |
| 机构建仓/增持 | ✅ | |
| 卖空比例下降 | ✅ | |
| 卖空数量下降 | ✅ | |
|  insider 买入 | ✅ | |

### 5.2 衍生品信号

| 信号 | 解读 |
|------|------|
| Call 大单买入 | 直接看涨押注 |
| Put 大量卖出 | 认为不会跌（看涨）|
| Put/Call Ratio < 0.7 | 情绪偏乐观 |
| Put/Call Ratio > 1.2 | 情绪偏悲观（反向指标）|
| IV 处于低位 + Call 买入 | 低成本看涨，概率高 |
| IV 处于高位 + Call 卖出 | 情绪过热，可能回调 |

### 5.3 市场情绪

| 维度 | 判断标准 |
|------|---------|
| **研报情绪** | 近期上调 vs 下调目标价的比例 |
| **新闻情绪** | 正面/负面报道占比 |
| **社区情绪** | 散户讨论热度，乐观 vs 悲观比例 |
| **反向指标** | 是否出现"全民看涨"的极端情绪？ |

## 工作流执行检查表

每次分析一只股票时，按以下顺序执行：

- [ ] **Step 1**：宏观方向是否支持这只股票的风格？
- [ ] **Step 2**：这只股票是否通过了初筛条件？
- [ ] **Step 3**：基本面是否出现拐点？成长驱动力是否清晰？
- [ ] **Step 4**：技术面是否处于突破/回踩/企稳状态？
- [ ] **Step 5**：资金面是否在流入？情绪面是否未过热？
- [ ] **Step 6**：是否制定了明确的买点、止损、目标、仓位？
- [ ] **Step 7**：是否设定了跟踪计划和退出触发条件？

> **只有当 Step 3-5 形成共振时，才考虑执行 Step 6。**

## 进阶技巧

### 如何提高胜率？

1. **等待回调**：突破后不要追高，等回踩确认
2. **分批建仓**：不要一次性满仓，分2-3批降低择时风险
3. **严格止损**：单笔亏损控制在账户的1-2%
4. **让利润奔跑**：达到目标后分批止盈，保留部分仓位博更大空间
5. **复盘记录**：每笔交易记录分析过程、执行情况和结果，持续优化

### 常见错误

| 错误 | 后果 | 避免方法 |
|------|------|---------|
| 只看技术面忽视基本面 | 买到"仙股反弹"或假突破 | 坚持基本面先行的筛选 |
| 基本面好但技术面破位时买入 | 被套在下跌中途 | 等待技术面企稳信号 |
| 没有止损计划 | 小亏变大亏 | 买入前必须设止损 |
| 情绪驱动交易 | 追高杀跌 | 按工作流执行，不被FOMO影响 |
| 仓位过重 | 单票爆仓风险 | 单票<10%，单笔风险<2% |
