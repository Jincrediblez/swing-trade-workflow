## 二、投研工作流框架（7步法）

### 2.1 整体框架

```
宏观趋势定方向 → 初筛池子 → 基本面深度筛选 → 技术面异动信号 → 资金/情绪共振验证 → 精确买卖点 → 仓位与风控
```

### 2.2 各步骤核心要点

| 步骤 | 目标 | 关键动作 |
|------|------|---------|
| **1. 宏观定方向** | 判断大盘周期+科技主线 | 查宏观数据、板块轮动、利率环境 |
| **2. 初筛股票池** | 缩到20-50只候选 | 成长性、市值、技术形态、RS Rating |
| **3. 基本面深筛** | 确认"真成长"+拐点 | 业绩拐点、成长驱动力、竞争壁垒、估值 |
| **4. 技术面精筛** | 基本面与技术面共振 | 趋势结构、量能、动量指标、支撑阻力 |
| **5. 资金+情绪共振** | 聪明钱进场？情绪不过热？ | 资金流向、期权异动、新闻情绪、社区情绪 |
| **6. 制定交易计划** | 买点、止损、目标、仓位 | 触发条件、分批建仓、止盈策略 |
| **7. 持仓跟踪风控** | 持续验证共振 | 每周检查清单、动态止损、退出触发器 |

### 2.3 核心原则

> **"共振"是核心**：单独基本面好但技术面破位 = 不着急买；单独技术面突破但基本面没改善 = 可能是假突破。
